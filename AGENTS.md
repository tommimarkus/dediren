# AGENTS.md

The canonical repository work guidance for AI coding agents lives in
[`CLAUDE.md`](CLAUDE.md). Read it first.

This file exists so agent tools that look for `AGENTS.md` are routed to the same
single source of truth. Do not duplicate guidance here; edit `CLAUDE.md`
instead.

For downstream agents that author Dediren JSON or run packaged bundles, see
[`docs/agent-usage.md`](docs/agent-usage.md).
